# Logic
## Conditionals
- If P then Q = P $\rightarrow$ Q (means P implies Q)
### Other Language for ${P}\rightarrow{Q}$
- *Q is necessary for P*
- *P only if Q*
- *When P, then Q*

| P   | Q   | P $\rightarrow$ Q |
| --- | --- | ----------------- |
| T   | T   | T                 |
| T   | F   | F                 |
| F   | T   | T                 |
| F   | F   | T                 |

## Examples
 - E.g. If P = 2 > 3 then Q = ${log}_3\pi$ = 2
	 - P, Q = *False, False*
	 - P $\rightarrow$ Q = *True*
