# Functions

- A function $f : A → B$ is:
## Injective
- if for all $a, a′ ∈ A$, $a \neq {a′}$ implies $f (a) \neq f (a′)$

## Surjective
-  if for every $b ∈ B$ there is an $a ∈ A$ with $f (a) = b$

## Bijective
- if $f$ is both *injective* and *surjective*.
